# D4 Helper - v2.1.0 릴리스 노트

> **배포일**: 2026년 1월 1일  
> **버전**: v2.1.0 (Event Tracker Edition)

## 🌟 하이라이트

이번 **v2.1.0** 업데이트는 디아블로 4의 주요 이벤트(지옥물결, 월드보스, 군단)를 실시간으로 추적하는 **이벤트 오버레이** 기능이 추가되었습니다.  
이제 게임 화면에서 눈을 떼지 않고도 다음 이벤트 시간을 정확하게 확인할 수 있습니다.

---

## ✨ 새로운 기능

### 1. 🕒 실시간 이벤트 오버레이 (Event Overlay)

- **지옥물결(Helltide)**, **월드보스(World Boss)**, **군단(Legion)** 이벤트 시간을 오버레이에 표시합니다.
- **초 단위 카운트다운**: 매초 줄어드는 타이머로 정확한 남은 시간을 제공합니다.
- **스마트 보스 정보**: "Azmodan" 등 구체적인 월드보스 이름을 표시합니다.
- **시각적 상태 알림**: 이벤트 진행 중(녹색), 대기 중(노란색/회색) 등 상태에 따라 색상이 변경됩니다.

### 2. 🎨 오버레이 디자인 개선

- **군단 이벤트 테마**: 군단 이벤트에 청록색/보라색 그라데이션의 전용 테마를 적용하여 시인성을 높였습니다.
- **가독성 향상**: 폰트 그림자와 글로우 효과를 강화하여 어두운 게임 화면에서도 잘 보이도록 개선했습니다.

---

## 🛠️ 개선 사항

### 🕷️ 크롤러 엔진 업그레이드

- **Helltides.com 대응**: 웹사이트 구조 변경에 맞춰 크롤러 로직을 전면 수정했습니다.
- **시간 형식 지원**: "1 hour, 20 minutes"와 같은 긴 시간 형식도 정확하게 파싱합니다.
- **랜덤 스크래핑**: 서버 부하를 줄이고 차단을 방지하기 위해 5~10분 간격의 랜덤 스크래핑을 도입했습니다.

### 📍 오버레이 위치 최적화

- **기본 위치 변경**: 오버레이 초기 위치를 프로필 레이아웃 우측(x: 610, y: 70)으로 변경하여 메인 UI와 겹치지 않도록 했습니다.
- **위치 리셋 기능**: 설정 파일 마이그레이션을 통해 잘못된 위치에 있던 오버레이를 자동으로 올바른 위치로 복구합니다.

---

## 🐛 버그 수정

- **[Fix]** 월드보스 시간이 "--:--"로 표시되던 파싱 오류 수정
- **[Fix]** 앱 재시작 시 오버레이 위치가 초기화되지 않던 문제 해결

# D4 Helper - v1.0.1 릴리스 노트

> **배포일**: 2025년 12월 7일  
> **버전**: v1.0.1 (Sanctuary Edition)

## 🌟 하이라이트

이번 **v1.0.1** 업데이트는 디아블로 4와의 호환성 개선과 사용자 경험 향상에 중점을 두었습니다.  
특히, 기존 키 입력 방식의 문제를 해결하기 위해 매크로 엔진을 전면 교체하여 안정성을 대폭 강화했습니다.

---

## ✨ 새로운 기능

### 1. ⚔️ 8개 스킬 슬롯 확장

- 최대 8개의 스킬을 동시에 자동화할 수 있습니다.
- 각 슬롯별로 **0ms ~ 5000ms**까지 정밀한 간격 설정이 가능합니다.

### 2. 🛑 스마트 일시정지 (Stop Keys)

- 게임 내 UI(인벤토리, 지도, 스킬창 등)를 열거나 채팅을 할 때 매크로가 **자동으로 일시정지**됩니다.
- 지원 키: `C`(인벤토리), `K`(스킬), `Tab`(지도), `M`(월드맵), `Enter`(채팅) 등

### 3. 💀 Hold to Pause (누르고 있으면 일시정지)

- 특정 키(기본값: `Shift`)를 누르고 있는 동안 매크로를 잠시 멈출 수 있습니다.
- 정밀한 조준이나 이동 시 유용하게 사용할 수 있습니다.

### 4. 🌍 다국어 지원 강화

- **한국어**, **English**, **日本語**, **简体中文** 4개 국어를 완벽하게 지원합니다.
- UI 언어가 즉시 변경되며, 상태 메시지도 해당 언어로 표시됩니다.

---

## 🛠️ 개선 사항

### 🚀 매크로 엔진 교체 (`robotjs` → `@nut-tree/nut-js`)

- 기존 `robotjs` 라이브러리가 DirectX 게임(디아블로 4)에서 키 입력을 제대로 전달하지 못하는 문제를 해결했습니다.
- 새로운 엔진은 **하드웨어 스캔 코드(Scan Code)**를 직접 전송하여 게임 내에서 완벽하게 인식됩니다.

### 🎨 UI/UX 디자인 리뉴얼

- 디아블로 4의 분위기를 살린 **Sanctuary 테마**를 적용했습니다.
- 고딕 스타일의 폰트와 다크 레드/골드 컬러 조합으로 몰입감을 높였습니다.
- 상태 표시(실행 중/정지/일시정지)가 더 직관적으로 변경되었습니다.

### 💾 설정 저장 시스템 개선

- `electron-store`를 도입하여 모든 사용자 설정이 로컬에 안전하게 자동 저장됩니다.
- 프로그램을 재시작해도 이전 설정(스킬 키, 간격, 옵션 등)이 그대로 유지됩니다.

---

## 🐛 버그 수정

- **[Critical]** 게임 내에서 키 입력이 무시되던 현상 수정 (DirectX 호환성 해결)
- **[Fix]** 언어 변경 시 상태 메시지("실행 중", "정지됨")가 번역되지 않던 문제 수정
- **[Fix]** 간격을 50ms 미만으로 설정 시 CPU 사용량이 급증하던 문제 최적화
- **[Fix]** 일부 시스템 단축키 충돌로 인해 글로벌 핫키가 작동하지 않던 문제 해결

---
