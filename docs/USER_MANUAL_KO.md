## 소개

### D4 Helper란?

D4 Helper는 디아블로 4 게임을 위한 스킬 자동 사용 도구입니다. 사용자가 설정한 키를 지정된 시간 간격으로 자동으로 입력하여, 반복적인 스킬 사용을 편리하게 만들어줍니다.

### 주요 기능

- ⚔️ **8개 스킬 슬롯**: 최대 8개의 스킬을 동시에 자동 실행
- ⏱️ **간격 조정**: 각 스킬마다 0~2000ms 간격 설정 가능
- 🎮 **Stop Keys**: 인벤토리, 맵 등 UI 열 때 자동 일시정지
- 🔥 **F1 단축키**: 빠른 시작/정지
- 🌍 **다국어 지원**: 한국어, 영어, 일본어, 중국어
- 🎨 **디아블로 테마**: 게임과 어울리는 다크 레드 UI

## 설치 방법

### 방법 1: 설치 프로그램 사용 (권장)

1. `D4 Helper Setup 2.1.0.exe` 다운로드
2. 실행 파일 더블클릭
3. 설치 마법사 안내에 따라 설치
4. 바탕화면 또는 시작 메뉴에서 실행

### 방법 2: 포터블 버전

1. `D4 Helper 2.1.0.exe` 다운로드
2. 원하는 폴더에 저장
3. 실행 파일 더블클릭하여 바로 실행

### 시스템 요구사항

- **운영체제**: Windows 10 (64-bit) 이상
- **메모리**: 최소 4GB RAM
- **저장공간**: 100MB 이상
- **기타**: .NET Framework 4.5 이상

## 기본 사용법

### Step 1: 스킬 키 설정

#### 1-1. 스킬 슬롯 선택

우측 패널의 **Skill Keys** 섹션에서 8개의 슬롯 중 하나를 선택합니다.

```
┌──────────────────────┐
│ 1 │  [1]  │  11 ms  │
│   ═══●═══════════    │
└──────────────────────┘
```

- **숫자 (1)**: 슬롯 번호
- **[1]**: 현재 설정된 키
- **11 ms**: 실행 간격 (밀리초)

#### 1-2. 키 입력

1. 키 입력 필드 (`[1]` 부분) 클릭
2. 원하는 키보드 키 누르기
3. 자동으로 키가 등록됩니다

**설정 가능한 키**:
- 숫자: 1, 2, 3, 4, 5, 6, 7, 8, 9, 0
- 문자: Q, W, E, R, A, S, D, F
- 특수키: Space, Tab, Shift
- 기능키: F1~F12

#### 1-3. 간격 조정

**방법 1: 슬라이더 사용**
- 빨간색 슬라이더를 좌우로 드래그
- 범위: 0~2000ms

**방법 2: 직접 입력**
1. 우측 상단 숫자 입력란 클릭
2. 원하는 값(ms) 입력
3. 범위: 0~5000ms

> 💡 **팁**: 일반 스킬은 1000ms, 빠른 스킬은 100ms 이하 추천

---

### Step 2: Stop Keys 설정 (선택사항)

좌측 패널의 **Stop Keys** 섹션에서 디아블로 UI 키를 토글할 수 있습니다.

```
┌─────────────────────────┐
│ Stop Keys               │
│ (Match Diablo Keys)     │
├─────────────────────────┤
│ 🎒 Inventory (C)    ○  │ ← OFF
│ ✕ Skills (K)        ●  │ ← ON
│ 👤 Follower (F)     ○  │
│ 🗺️ Map (Tab)        ○  │
│ 🌍 World Map (M)    ○  │
│ 🌀 Town Portal(T)   ○  │
│ 💬 Chat (Enter)     ○  │
│ 💬 Whisper (/)      ○  │
└─────────────────────────┘
```

#### 토글 방법

각 항목 우측의 **원형 버튼** 클릭:
- **○ (회색)**: 비활성화
- **● (빨간 불빛)**: 활성화

#### 동작 방식

활성화된 키를 누르면:
1. 매크로가 **자동 일시정지**
2. 해당 키를 다시 누르면 (UI 닫히면) **매크로 재개**

**예시**:
- `Inventory (C)` 활성화
- 게임에서 `C` 키 눌러 인벤토리 열기
- → 매크로 자동 정지
- 인벤토리 닫기 (다시 `C`)
- → 매크로 자동 재개

---

### Step 3: 매크로 시작/정지

#### 시작 방법

**방법 1: 버튼 클릭**
1. 좌측 패널의 **🔥 Start Key (F1)** 버튼 클릭
2. 버튼이 빛나기 시작하면 실행 중

**방법 2: 단축키 (권장)**
1. 키보드에서 **F1** 키 누르기
2. 게임 중에도 사용 가능 (글로벌 단축키)

#### 정지 방법

**방법 1: 버튼 클릭**
- 좌측 패널의 **⬛ Stop Key (F1)** 버튼 클릭

**방법 2: 단축키 (권장)**
- 키보드에서 **F1** 키 다시 누르기

#### 상태 확인

버튼 아래에 상태 표시등이 있습니다:

| 색상 | 상태 | 설명 |
|------|------|------|
| 🟢 녹색 (깜빡임) | 실행 중 | 매크로가 정상 작동 중 |
| 🔴 빨강 | 정지됨 | 매크로가 정지된 상태 |
| 🟡 노랑 | 일시정지 | Stop Key로 일시정지됨 |

## 고급 기능

### Special Key (Hold to Pause)

특정 키를 **누르고 있는 동안만** 매크로를 일시정지하는 기능입니다.

#### 설정 방법

1. 하단의 **Special Key (Hold to Pause)** 패널 찾기
2. `Key:` 입력란 클릭
3. 원하는 키 입력 (예: Shift, Ctrl, Alt)

```
┌─────────────────────────────────┐
│ Special Key (Hold to Pause)     │
├─────────────────────────────────┤
│ 💀 Key: [Shift]                 │
│ ⚠️ 경고: 이 키를 누르고 있으면   │
│    매크로가 일시정지됩니다.      │
└─────────────────────────────────┘
```

#### 사용 예시

**상황**: 정밀 조준이 필요한 순간
1. Special Key를 `Shift`로 설정
2. 매크로 실행 중 `Shift` 키 누름
3. → 매크로 일시정지
4. 수동으로 조준하여 스킬 사용
5. `Shift` 키를 뗌
6. → 매크로 자동 재개

### 설정 프로필 관리

#### 설정 저장

1. 상단 메뉴바의 **💾** (저장) 아이콘 클릭
2. 현재 설정이 자동으로 저장됩니다

**저장되는 내용**:
- 8개 스킬 슬롯 설정 (키 + 간격)
- Stop Keys 활성화 상태
- Special Key 설정
- 언어 설정

#### 설정 불러오기

1. 상단 메뉴바의 **📄** (불러오기) 아이콘 클릭
2. 이전에 저장된 설정이 복원됩니다

> 💡 **자동 저장**: 프로그램 종료 시 현재 설정이 자동으로 저장됩니다.

## 문제 해결

### 문제 1: 매크로가 동작하지 않음

**증상**: F1을 눌러도 아무 반응 없음

**해결 방법**:
1. 스킬 슬롯에 키가 제대로 설정되어 있는지 확인
2. 간격이 0이 아닌지 확인
3. 프로그램을 **관리자 권한**으로 실행
   - 실행 파일 우클릭 → "관리자 권한으로 실행"
4. Windows 방화벽이나 백신 프로그램 확인

### 문제 2: 일부 키만 입력됨

**증상**: 설정한 8개 중 일부만 동작

**해결 방법**:
1. 각 슬롯의 **활성화 여부** 확인
2. 슬롯에 키가 비어있지 않은지 확인 (`Empty` 표시)
3. 간격이 너무 길지 않은지 확인

### 문제 3: Stop Keys가 작동하지 않음

**증상**: 인벤토리를 열어도 매크로가 멈추지 않음

**해결 방법**:
1. Stop Keys 토글이 **● (빨간불)**로 활성화되어 있는지 확인
2. 일부 키(Tab, Enter)는 Windows 제약으로 작동하지 않을 수 있음
3. 대신 다른 Stop Key 사용 권장

### 문제 4: 프로그램이 느려짐

**증상**: 프로그램 반응이 느리거나 버벅임

**해결 방법**:
1. 너무 짧은 간격(10ms 이하) 사용 시 CPU 부하 증가
2. 최소 50ms 이상 권장
3. 활성화된 슬롯 수를 줄이기 (4~5개 권장)
4. 다른 프로그램 종료하여 리소스 확보

### 문제 5: 게임에서 감지됨

**증상**: 게임 내 경고 또는 계정 정지

**해결 방법**:
⚠️ **이 프로그램은 소프트웨어 레벨 매크로로, 게임 보안 시스템에 감지될 수 있습니다.**

- 교육 및 개인 목적으로만 사용
- 공식 게임 서버 사용 시 위험 감수
- 대안: 하드웨어 매크로 키보드/마우스 사용 고려

## 자주 묻는 질문

### Q1: 최대 몇 개의 스킬을 동시에 사용할 수 있나요?

**A**: 최대 8개의 스킬을 동시에 자동 실행할 수 있습니다.

### Q2: 최소/최대 간격은 얼마인가요?

**A**:
- 슬라이더: 0~2000ms
- 직접 입력: 0~5000ms
- 권장: 50ms 이상 (시스템 부하 방지)

### Q3: F1 대신 다른 단축키를 사용할 수 있나요?

**A**: 현재 버전에서는 F1 고정입니다. 향후 업데이트에서 사용자 정의 가능 예정.

### Q4: 다른 게임에서도 사용할 수 있나요?

**A**: 네, 키보드 입력을 지원하는 모든 게임/프로그램에서 사용 가능합니다.

### Q5: 설정 파일은 어디에 저장되나요?

**A**:
- Windows: `%APPDATA%\d4-helper\config.json`
- 경로: `C:\Users\[사용자명]\AppData\Roaming\d4-helper\`

### Q6: 매크로 사용이 안전한가요?

**A**:
⚠️ **주의**: 이 프로그램은 교육 목적으로 제작되었습니다.
- 소프트웨어 레벨 매크로로 감지 가능
- 게임 이용약관 위반 가능성
- 사용자 책임 하에 사용

### Q7: 프로그램이 업데이트되나요?

**A**: GitHub 릴리즈 페이지에서 최신 버전 확인 가능합니다.

### Q8: 맥(Mac)이나 리눅스에서도 되나요?

**A**: 현재 Windows 전용입니다. 다른 OS 지원은 계획 중입니다.

## 주의사항

### ⚠️ 법적 책임

1. **게임 이용약관**
   - 대부분의 온라인 게임은 매크로 사용을 금지합니다
   - 계정 정지 또는 영구 차단 위험 존재

2. **사용자 책임**
   - 이 프로그램 사용으로 인한 모든 결과는 사용자 책임입니다
   - 개발자는 어떠한 법적 책임도 지지 않습니다

3. **권장 용도**
   - 교육 및 학습 목적
   - 개인 프로젝트 테스트
   - 오프라인 게임 또는 허용된 환경

### 🛡️ 보안 주의사항

1. **관리자 권한**
   - 이 프로그램은 키보드 입력을 위해 관리자 권한 필요
   - 신뢰할 수 있는 소스에서만 다운로드

2. **백신 경고**
   - 일부 백신 프로그램이 오탐지할 수 있음
   - 키보드 후킹 기능 때문에 발생
   - 신뢰할 수 있는 버전 사용

3. **개인정보**
   - 이 프로그램은 개인정보를 수집하지 않습니다
   - 설정 파일은 로컬에만 저장

### 💡 사용 팁

1. **처음 사용 시**
   - 간격을 길게 설정하고 테스트 (1000ms+)
   - 노트패드 등에서 먼저 테스트
   - 게임에서 사용 전 충분히 숙지

2. **효율적 사용**
   - 자주 사용하는 스킬만 활성화
   - Stop Keys 적절히 활용
   - 정기적으로 설정 저장

3. **문제 발생 시**
   - 프로그램 재시작
   - 관리자 권한으로 실행
   - 설정 초기화 (설정 파일 삭제)
